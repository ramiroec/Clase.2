Atributos Especiales de Formularios HTML

Saltar Validación con formnovalidate

HTML permite omitir la validación de formularios usando el atributo formnovalidate.

Tip: Esto es útil cuando quieres permitir al usuario enviar un formulario aunque algunos campos no cumplan con las validaciones estándar.

Información Personal 

<form>
  <fieldset>
    <legend>Información Personal</legend>
    <label for="nombre">Nombre:</label>
    <input type="text" name="nombre" required><br>
    <label for="email">Email:</label>
    <input type="email" name="email" required><br>
  </fieldset>
  
  <input type="submit" value="Enviar con validación">
  <input type="submit" formnovalidate value="Enviar sin validar">
</form>

Atributos: readonly, autofocus y hidden

readonly

Especifica que un campo es de solo lectura (no se puede modificar).

autofocus

Coloca automáticamente el foco en el campo cuando se carga la página.

hidden

Oculta el campo (útil para enviar datos que el usuario no necesita ver).

Información Personal 

<form>
  <fieldset>
    <legend>Información Personal</legend>
    <input type="hidden" name="id" value="1">
    
    <label for="nombre">Nombre:</label>
    <input type="text" name="nombre" autofocus><br>
    
    <label for="email">Email:</label>
    <input type="email" name="email" value="usuario@ejemplo.com" readonly><br>
  </fieldset>
  
  <input type="submit">
</form>

Elemento <select> para listas desplegables

Permite elegir un valor de una lista predefinida de opciones.

<select id="frutas" name="frutas">
  <option value="manzana">Manzana</option>
  <option value="naranja">Naranja</option>
  <option value="uva">Uva</option>
  <option value="plátano">Plátano</option>
  <option value="fresa">Fresa</option>
</select>

Tip: Puedes usar el atributo selected en un <option> para preseleccionar un valor.
