Guía Visual de Campos <input> en HTML

¿Qué es un campo <input>?

Los campos <input> permiten crear diferentes tipos de controles interactivos en formularios web, como:

  • Cuadros de texto
  • Selectores de fecha
  • Campos numéricos
  • Contraseñas
  • Botones
  • Y muchos más...

Los campos se validan automáticamente al presionar el botón enviar.

Tipos de Campos Input

<input type="color">

<input type="date">

<input type="email">

<input type="number">

<input type="password">

<input type="range">

<input type="text">

<input type="time">

<input type="url">

Validación con Min y Max

Puedes definir valores máximos y mínimos para campos numéricos y de fecha.

Ejemplo:





<form action="ver.php">
  Edad: <input type="number" name="edad" min="1" max="100"><br>
  Hora: <input type="time" name="hora"><br>
  <input type="submit">
</form>

Variaciones de Campos de Fecha
