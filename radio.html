Checkboxes vs Radio Buttons en HTML

Checkboxes (Casillas de verificación)

Características: Permiten seleccionar múltiples opciones de una lista. Cada checkbox es independiente de los demás.

Seleccione una o varias opciones: 
<form>
  <strong>Seleccione una o varias:</strong>
  <br>
  <input type="checkbox" id="claro"><label for="claro">Claro</label>
  <input type="checkbox" id="tigo"><label for="tigo">Tigo</label>
  <input type="checkbox" id="vox"><label for="vox">Vox</label>
  <input type="submit">
</form>

Radio Buttons (Botones de opción)

Características: Permiten seleccionar solo una opción de un conjunto. Todos los radio buttons con el mismo name están agrupados.

Seleccione una única opción: 
<form>
  <strong>Seleccione una opción:</strong>
  <br>
  <input type="radio" id="1" name="X"><label for="1">Claro</label>
  <input type="radio" id="2" name="X"><label for="2">Tigo</label>
  <input type="radio" id="3" name="X"><label for="3">Vox</label>
  <input type="submit">
</form>

Comparación: Checkboxes vs Radio Buttons

Checkboxes

  • Permiten selección múltiple
  • Cada opción es independiente
  • Usar cuando el usuario puede elegir cero, una o más opciones
  • No necesitan compartir el mismo name

Radio Buttons

  • Permiten selección única
  • Opciones son mutuamente excluyentes
  • Usar cuando el usuario debe elegir exactamente una opción
  • Deben compartir el mismo name para agruparse

Consejo: Usa <label> asociado con for e id para mejorar la accesibilidad y hacer que sea más fácil hacer clic en las opciones.
