Atributos y Validación de Formularios HTML

Input de Rango (<input type="range">)

Permite elegir entre varios valores de forma visual mediante un control deslizante.

5 
<input type="range" min="0" max="10" step="2" value="5">
min

Valor mínimo permitido (0 por defecto)

max

Valor máximo permitido (100 por defecto)

step

Incremento entre valores (1 por defecto)

value

Valor inicial del control

Validación y Mensajes de Error Personalizados

* Campo obligatorio * Mínimo 6 caracteres * Entre 18 y 99 años 
<form>    
  Nombre: <input type="text" required
    oninvalid="setCustomValidity('Nombre es obligatorio')"
    oninput="setCustomValidity('')"><br>
    
  Password: <input type="password" required pattern=".{6,}"
    oninvalid="setCustomValidity('Mínimo 6 caracteres')"
    oninput="setCustomValidity('')"><br>
    
  Edad: <input type="number" min="18" max="99"
    oninvalid="setCustomValidity('Edad entre 18 y 99 años')"
    oninput="setCustomValidity('')"><br>
    
  <input type="submit">
</form>

Atributos Comunes de Input

type

Indica el tipo de dato (text, number, email, password, etc.)

value

Establece el valor inicial del campo

required

Indica que el campo es obligatorio

placeholder

Muestra un texto que desaparece al escribir

disabled

Deshabilita el campo (no se puede interactuar)

readonly

Hace que el campo no sea modificable

pattern

Expresión regular para validación personalizada

min/max

Valores mínimo y máximo para números/fechas

Tip: Combina estos atributos para crear formularios robustos con validación automática del navegador.
